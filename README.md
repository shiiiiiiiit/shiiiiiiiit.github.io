# 项目背景

这是一个基于 WebRTC P2P 的多人在线坦克大战游戏，使用纯前端技术（ES Modules）。

---

# 项目结构

## 📁 根目录
- `index.html` - 入口HTML，包含所有界面结构（含静音按钮）
- `css/style.css` - 全部样式（含音效控制按钮样式）

## 📁 js/ 核心模块
- `main.js` - 应用入口，绑定事件，初始化资源加载（图片+音效）
- `constants.js` - 所有游戏常量（地图大小、速度、伤害、音效列表等）
- `helpers.js` - 纯工具函数（无依赖）
- `state.js` - 全局状态管理
- `utils.js` - 依赖 state 的工具函数

## 📁 js/assets/
- `images.js` - 图片资源加载与缓存
- `sounds.js` - 音效资源加载、播放、静音控制

## 📁 js/systems/ 底层系统
- `noise.js` - Perlin 噪声（地形生成）
- `ground.js` - 地面分块渲染系统
- `spatialGrid.js` - 空间分区（优化碰撞检测）
- `objectPool.js` - 对象池
- `collision.js` - 碰撞检测函数

## 📁 js/game/ 游戏逻辑
- `map.js` - 地图生成（墙壁、障碍物、出生点）
- `buffs.js` - Buff 系统（生成、拾取、应用效果）
- `items.js` - 道具系统（拾取、使用、投掷物）
- `laser.js` - 激光武器系统
- `shrink.js` - 缩圈系统
- `movement.js` - 移动与冲刺系统（点按释放触发短冲刺，长按超过阈值立即触发长冲刺）
- `shooting.js` - 射击系统（含射击音效播放）
- `entities.js` - 子弹、怪物更新
- `gameLoop.js` - 主循环

## 📁 js/network/
- `communication.js` - P2P 通信（房间、消息同步）

## 📁 js/ui/
- `screens.js` - 界面管理（加入、大厅、结算）

## 📁 js/render/ 渲染模块
- `renderer.js` - 渲染入口
- `renderEntities.js` - 实体渲染（坦克、怪物、道具等）
- `renderEffects.js` - 特效渲染（激光、爆炸、烟雾）
- `renderUI.js` - UI 渲染（HUD、小地图、准星）
- `renderUtils.js` - 渲染工具函数

## 📁 assets/
- 图片资源（坦克、子弹、特效、地形、UI图标等 PNG 文件）

## 📁 assets/sounds/
- 音效资源 （MP3 文件）
